**LaunchAgent.ai** is an automated co-founder designed to bridge the gap between a raw idea and a market-ready startup. Built using the **"Vibe Coding"** philosophy, this agent synthesizes market strategy, real-time domain availability, visual identity, legal documentation, and technical roadmaps in under 60 seconds.

---

## 📽️ Project Overview

Founders often waste weeks on "peripheral friction"—naming, branding, legal paperwork, and roadmapping—before writing a single line of core logic. **LaunchAgent.ai** provides a centralized AI Workspace that generates a 360° launch kit, allowing founders to focus purely on the **Vibe** and the **Code**.



---

## 🛠 The "Vibe" Stack

This project was built using a high-velocity development cycle, leveraging AI at every layer of the stack to ensure rapid iteration and feature density.

* **Core Engine:** [React.js](https://reactjs.org/) (Vite)
* **AI Brain:** [Google Gemini 1.5 Flash API](https://ai.google.dev/) (Strategic logic & Content synthesis)
* **Domain Intelligence:** [WHOIS XML API](https://www.whoisxmlapi.com/) (Real-time domain availability & registration status)
* **Vibe Tools:** [VS Code](https://code.visualstudio.com/), [GitHub Copilot](https://github.com/features/copilot), [Cursor](https://www.cursor.com/)
* **Authentication:** [Auth0](https://auth0.com/)
* **Styling:** [Tailwind CSS](https://tailwindcss.com/) & [Framer Motion](https://www.framer.com/motion/)
* **Assets:** [Lucide React](https://lucide.dev/), [JSZip](https://stuk.github.io/jszip/) (Client-side bundling)

## ✨ Key Features

### 🔍 Real-Time WHOIS Domain Search
Integrates live domain lookup to verify if your AI-generated brand names are actually available. No more falling in love with a name that is already taken.

### 🎯 Strategic Intelligence
AI-evaluated **Launch Readiness Scores**. We don't just generate text; we calculate brand clarity, messaging strength, and market risk using a proprietary scoring algorithm.

### 🎨 Visual Identity Studio
Generates a complete Brand Kit including HEX color palettes, typography suggestions, and high-fidelity **AI Logo Prompts** for Midjourney/DALL-E.

### 📄 Technical & Legal Suite
Automated generation of GitHub-ready assets:
* `README.md` (Technical architecture)
* `PRIVACY.md` (GDPR-compliant policies)
* `TOS.md` (Terms of Service)
* `SECURITY.md` (Security protocols)

### 🌐 MVP Preview Mode
A live "Simulated Browser" that renders a dynamic landing page, features dashboard, and pricing table specifically for your project's unique data.

### 🚀 7-Day Execution Roadmap
A gamified, day-by-day checklist that breaks down the launch process into manageable, industry-specific tasks.

## 🧪 Vibe Coding Methodology

This project was developed using a **Vibe Coding** approach:
1.  **Contextual Seeding:** Providing the AI with deep project context to ensure coherent feature development across disparate modules.
2.  **Iterative Refinement:** Rapidly looping through UI/UX improvements via Copilot and Gemini Chat.
3.  **JSON Architecting:** Utilizing structured JSON prompting to ensure the AI serves as a data-provider, enabling the frontend to remain dynamic and reactive.

## ⚙️ Installation & Setup

1.  **Clone the Repo:**
    ```bash
    git clone url
    cd launchagent
    ```

2.  **Install Dependencies:**
    ```bash
    npm install
    ```

3.  **Environment Variables:**
    Create a `.env` file in the root directory:
    ```env
    VITE_GOOGLE_GENAI_API_KEY=your_gemini_api_key
    VITE_WHOIS_API_KEY=your_whois_api_key
    VITE_AUTH0_DOMAIN=your_auth0_domain
    VITE_AUTH0_CLIENT_ID=your_auth0_client_id
    ```

4.  **Launch the App:**
    ```bash
    npm run dev
    ```

## 📦 Exporting Assets

Once you’ve generated your startup kit, use the **"Export All Assets"** button. This uses client-side compression to bundle your strategy, roadmap, and markdown documentation into a single `.zip` file, ready for immediate deployment into a fresh repository.
